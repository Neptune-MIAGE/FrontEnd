# Installation du projet

1. Cloner le dépot

2. Se placer à la racine du projet dans le terminal et lancer les commandes suivantes :

```sh
npm install
npm update
npm run dev
```

Entrez ensuite l'URL donné dans vore navigateur pour accéder au site.
